# Weekly Review & Planning

Comprehensive weekly review and next week planning. Run every Friday afternoon or Monday morning.

## Instructions

When the user runs `/weekly`, generate a complete weekly review and planning session and **save it as a file**.

**File Location:** `/Users/elbowspeak1/Documents/Obsidian Vault/Doikayt/Weekly/YYYY-MM-DD.md` (dated for the week ending date)

**Important:** Always use the Write tool to create the weekly review file so the user can reference it in Obsidian.

## Analysis Process

### 1. **Review Last 7 Days**
- Check daily notes from the past week
- Identify what was accomplished
- Note what was planned but not completed
- Track patterns (productive days, bottlenecks)

### 2. **Portfolio Assessment**
- Status of all 7 companies
- Board meetings this week (completed or upcoming)
- Key updates or decisions
- Actions taken vs. planned
- Trends across portfolio

### 3. **Relationship Management**
- Who did you connect with?
- Who did you miss connecting with?
- New connections made
- Introductions facilitated
- Network health trends

### 4. **Consulting/Revenue Work**
- Projects progressed
- Client deliverables completed
- Revenue generated or invoiced
- Upcoming client commitments
- Pipeline development

### 5. **Strategic Initiatives**
- Ideas explored or validated
- Research completed
- New opportunities identified
- Long-term projects advanced

### 6. **PAI Integration Updates**
- Run weekly PAI repository check
- Review new features/patterns available
- Identify 1-2 features to integrate this week
- Update automation systems

## Analysis Logic

### **Scan Past Week**

**Daily Notes Pattern:**
Look for files matching date pattern from last 7 days:
- Count how many exist (daily note habit)
- Extract todos and check completion
- Identify recurring themes or mentions
- Track which days were productive

**Modified Files:**
Check what files were modified in last 7 days:
- Portfolio company updates
- Contact interactions
- Project progress
- New ideas or research

### **Calculate Metrics**

**Portfolio Engagement:**
- Which companies' files were modified?
- How many days since each was touched?
- Board meetings or major events?

**Relationship Activity:**
- Contacts added or updated
- Last_contact dates in last 7 days
- New introductions made

**Time Allocation (estimate):**
- Based on files modified and content
- Consulting hours if tracked
- Meeting notes and interactions

### **Trend Analysis**

**Week-over-Week:**
- More/less portfolio engagement?
- Network activity up/down?
- Revenue work consistent?
- Strategic work happening?

**Patterns:**
- Which day of week most productive?
- What types of work getting done?
- What's being neglected?

### **Forward Planning**

**Intelligent Suggestions:**

If portfolio company needs attention:
- "ConfigHub board meeting next Thursday - start prep Wednesday"

If relationship health declining:
- "3 strong relationships dormant > 60 days - prioritize reconnection"

If consulting revenue low:
- "Only X hours this week - pipeline concern? Client check-ins needed?"

If no strategic work:
- "No idea/research progress - schedule thinking time"

**PAI System Check:**
- Run `bash Scripts/automation/check-pai-updates.sh` to check for new PAI features
- Review the generated report at `Daily/PAI-Updates-YYYY-MM-DD.md`
- Suggest 1-2 new features to integrate based on current workflow needs

## Interactive Elements

**Ask questions to make planning specific:**
- "What's your #1 goal for next week?"
- "Which portfolio company needs most attention?"
- "How many consulting hours committed next week?"
- "Any big meetings or deadlines to plan around?"

**Offer tailored suggestions:**
- Based on last week's accomplishments
- Based on gaps or missed items
- Based on upcoming events
- Based on portfolio status

## Important Notes

- Be honest but constructive about incomplete items
- Celebrate wins explicitly
- Provide specific, actionable next steps
- Balance reflection with forward planning
- Adapt tone: end-of-week = reflective, start-of-week = motivational
- Keep metrics simple and actionable
- Connect weekly priorities to longer-term goals
- Suggest one habit improvement per week
- End with clear, confidence-building next actions

## Usage Pattern

**Best Times:**
- **Friday afternoon (3-5pm):** Week winding down, reflective mindset
- **Monday morning (9-10am):** Fresh start, planning mindset

**Frequency:**
- Weekly (ideally Friday)
- Can run Monday if Friday was missed
- Can run mid-week for course correction

**Paired With:**
- Run `/portfolio-update` before `/weekly` if portfolio focus week
- Run `/relationship-health` monthly in addition to weekly
- Run `/daily` every other day minimum for best weekly review data
