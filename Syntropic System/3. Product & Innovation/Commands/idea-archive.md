# Idea Archive

Move validated ideas from "someday/maybe" to active execution with concrete project plans.

## Instructions

When the user runs `/idea-archive [idea]`, create a comprehensive execution plan and move the idea from planning to action.

## Usage

```
/idea-archive AI-powered portfolio reporting tool
/idea-archive content series on agentic AI
/idea-archive consulting service for early-stage startups
/idea-archive [idea that's been validated and matched]
```

## Purpose

Ideas die in the "validated" stage. This command:
- Creates concrete execution plan with milestones
- Moves idea to `Ideas/In_Execution/` folder
- Tracks progress from start to launch
- Ensures follow-through (not just planning)

**Use this command when:**
- Idea scored >20 in `/idea-validate`
- Found strong matches in `/idea-match`
- You're committed to executing (not just exploring)

## Execution Framework

### 1. **Define Success**
- What does "done" look like?
- How do you measure success?
- What's the minimum viable version?

### 2. **Break Down Phases**
- Research/Validation phase
- Build/Create phase
- Launch/Ship phase
- Iterate/Scale phase (if applicable)

### 3. **Create Timeline**
- Realistic milestones with dates
- Weekly commitments
- Dependencies and blockers

### 4. **Assign Resources**
- Who's involved (including you)
- What tools/systems needed
- Budget allocated

### 5. **Track Progress**
- Weekly check-ins (built into `/daily`)
- Milestone completion tracking
- Pivot signals (when to adjust)

- **Execution beats planning:** Don't over-plan, start building

- **Kill bad projects:** Better to kill early than fail late
- **Share progress:** Update network, create content, build in public
